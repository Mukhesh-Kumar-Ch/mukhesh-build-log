# Mukhesh-Build-Log

> "Sometimes you gotta run before you can walk."

This repository is my personal engineering log.

I explore a lot of things: AI, software, developer tools, startups, research papers, random technologies, and occasionally ideas that probably shouldn't work.

Instead of letting those experiments disappear into browser history, I document them here.

Inside you'll find:

* Things I learned
* Tools I tested
* Projects I built
* Certifications I'm preparing for
* Research rabbit holes I fell into
* Failures, confusion, and lessons learned

This isn't a portfolio.

It's the workshop behind the portfolio.

A record of curiosity, continuous learning, and the process of becoming a better engineer.
